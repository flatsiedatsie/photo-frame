    Receive photos via Localsend 

    Localsend is a popular open source app for mobile phones and laptops that allows for easy sharing of files between your devices. Learn more

    Localsend is like Apple's Airdrop, but it's available for all phones, laptops and other devices. To use it you have to install the LocalSend app on your phone or laptop.

    While Localsend it switched on, any device on your network that also has Localsend installed can upload photos to Photo Frame. When you're done, simply turn Localsend off again.

    You can add an optional security pin code in this addon's settings. With that enabled the pin code must be entered before the Photo Frame will accept new files.

    Learn more at localsend.org

     

    Privacy mode

    Select some photos that are 'safe' to show no matter who your guests are. Then you can enable privacy mode, which will only show those 'safe' photos for a while.

    During that time there will be no indication that privacy mode is active, and it will look like the 'unsafe' photos have been deleted. But don't worry, they're only hidden.
